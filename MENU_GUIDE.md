# 選單導覽指南 / Menu Navigation Guide

## 應用程式選單結構

本指南展示新增功能在應用程式選單中的位置。

```

├── 檔案 (File)
│   └── 結束 (Exit)
│
├── 檢視 (View) ⭐ 新增！
│   └── 修改紀錄 / 版本歷史 (Version History) ⭐ 新增！
│       └── 開啟修改紀錄視窗，顯示最近 100 筆變更記錄
│
├── 設定 (Settings) ⭐ 新增！
│   ├── 更改背景圖片 (Change Background) ⭐ 新增！
│   │   └── 開啟檔案選擇對話框，選擇背景圖片
│   │
│   └── 重設背景 (Reset Background) ⭐ 新增！
│       └── 恢復預設背景樣式

    └── 關於 (About)
```

## 功能使用流程

### 📝 查看修改紀錄

1. 點選選單列的 **「檢視」**
2. 點選 **「修改紀錄 / 版本歷史」**
3. 修改紀錄視窗開啟，顯示：
   - 時間
   - 類型 (event/task)
   - 操作 (新增/更新/刪除)
   - 實體ID
   - 變更欄位
   - 詳細資訊
4. 點選任意記錄列，查看完整的變更詳情
5. 點選 **「重新整理」** 按鈕載入最新記錄
6. 點選 **「關閉」** 按鈕關閉視窗

### 🎨 更換背景圖片

1. 點選選單列的 **「設定」**
2. 點選 **「更改背景圖片」**
3. 檔案選擇對話框開啟
4. 瀏覽並選擇圖片檔案（支援 PNG, JPG, JPEG, BMP, GIF）
5. 點選 **「開啟」** 確認選擇
6. 背景圖片立即套用
7. 成功訊息對話框顯示

### 🔄 重設背景

1. 點選選單列的 **「設定」**
2. 點選 **「重設背景」**
3. 確認訊息對話框顯示
4. 背景恢復為預設樣式

## 鍵盤快捷鍵建議（未來可實作）

目前版本暫不支援鍵盤快捷鍵，但建議未來可以新增：

- `Ctrl+H` - 開啟修改紀錄
- `Ctrl+B` - 更改背景圖片
- `Ctrl+Shift+B` - 重設背景

## 常見問題 (FAQ)

### Q: 修改紀錄可以保存多久？
A: 所有修改紀錄永久保存在本地資料庫中，預設顯示最近 100 筆。

### Q: 背景圖片會不會影響效能？
A: 不會。圖片僅在啟動時載入一次，對效能影響極小。

### Q: 可以使用線上圖片嗎？
A: 目前僅支援本地圖片。線上圖片功能可在未來版本中新增。

### Q: 背景圖片儲存在哪裡？
A: 僅儲存圖片的檔案路徑，不複製圖片檔案。請勿刪除或移動已設定的圖片。

### Q: 修改紀錄可以匯出嗎？
A: 目前版本不支援匯出功能，但可在未來版本中新增 CSV 或 JSON 匯出。
